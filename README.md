# CashoutService.my Landing Page

Project landing page modern menggunakan Next.js App Router, TypeScript, dan Tailwind CSS v4.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 (configured in globals.css)
- **Icons:** Lucide React
- **Animation:** Framer Motion (used sparsely)
